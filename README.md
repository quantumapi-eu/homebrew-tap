# QuantumAPI Homebrew Tap

Official Homebrew tap for [QuantumAPI](https://quantumapi.eu) CLI tools.

## Installation

```bash
brew tap quantumapi-eu/tap
brew install qapi
```

## Upgrade

```bash
brew upgrade qapi
```
